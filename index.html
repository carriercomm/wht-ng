     Web Hosting Toolkit (WHT) is a software for maintaining commercial (PayPal) or free (with header, footer and pop-up) webhosting server. It can create system user for ftp access, create many virtual domains and subdomains owned by the user, create many virtual email accounts for every domain, create MySQL databases, view traffic statistic ... For full info see the features page.

There are many Control Panels of this type. Why you should prefer Web Hosting Toolkit?
  • WHT is intended to be vastly configurable and distribution independent. If you install the required software, you can use the program under any Linux distribution.

  • WHT is absolutely free.

  • If you are a web developer, probably you have struggled the configuration limitations of of some hosting company's server. If you are thinging over the idea of starting your own server offering hosting to your clients at low price, WHT is just for you.

  • Web Hosting Toolkit offers multilanguage support. If your language is not in the list you can translate WHT yourself.

  • You can manage the content of WHT.

  • You can take advantage of all the profits Qmail offers.
